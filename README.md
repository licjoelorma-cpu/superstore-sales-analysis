# Superstore Sales Analysis

## Descripción

Proyecto de análisis exploratorio de datos (EDA) enfocado en evaluar el desempeño de ventas y rentabilidad de una tienda retail utilizando Python.

El análisis busca identificar patrones de comportamiento, productos con pérdidas, impacto de descuentos y diferencias regionales para generar insights de negocio.

---

## Objetivo

Analizar ventas, ganancias y descuentos para identificar oportunidades de mejora en rentabilidad y desempeño comercial.

---

## Dataset

Dataset de Superstore obtenido desde Kaggle.

---

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
- GitHub

---

## Principales insights

- Technology fue la categoría con mayor rentabilidad.
- Tables presentó las mayores pérdidas entre las subcategorías analizadas.
- Se identificó una relación negativa entre descuentos elevados y rentabilidad.
- West mostró el mejor desempeño en ventas y ganancias.
- Central presentó menores márgenes de rentabilidad en comparación con otras regiones.
